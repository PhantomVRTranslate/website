Project Phantom

Packaged 3D Components

A project envisioned to be a node package module (npm) that allows developers to create web applications in 3D using React VR, a framework for building VR projects using the declarative power of React.

See Package

Full Documentation

Each component, whether it be text, a video, carousel, or a gallery, among others, is easy to use and is capable of piecing together your application using React VR.
